### Effective Image Processor

OpenCV를 활용한 C++ Native Image Viewer dll 입니다.

# Environment
- **Visual Studio 2022**
- **OpenCV 4.12.0**
- **C++ 20**
- **.NET Framework 4.8**

# Project
- **ImageCore** : C++(Win32) Native Image Viewer dll 프로젝트입니다. (* ImageCore.dll)
- **ImageCoreTester** : C++(Win32)로 작성된 테스트 프로젝트입니다. 
- **ImageCoreWrapper** : C++/CLI 래핑 프로젝트입니다. (* ImageCoreWrapper.dll)
- **ImageCoreTester_WPF** : C# WPF로 작성된 테스트 프로젝트입니다.
